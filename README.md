## 📝 Додаток для створення та перегляду квізів (Quiz App)

Сучасний fullstack-додаток для створення інтерактивних вікторин та тестів. Проект підтримує три типи запитань: True/False, текстові відповіді (input) та варіанти з множинним вибором (checkbox). Інтерфейс повністю адаптований під мобільні пристрої та ПК.

## 🛠 Технологічний стек

- **Frontend:** Next.js (App Router), React Hook Form, Zod, CSS Modules
- **Backend:** Node.js, Express, TypeScript, Prisma ORM, SQLite
- **Якість коду:** ESLint, Prettier

## 🚀 Як запустити проект локально

```bash
git clone https://github.com/podyryakoanatoliy/quiz-builder.git
cd quiz-builder
```

# 1. Налаштування та запуск Бекенду
   Перейдіть у папку з бекендом командою: 
   ```bash
   cd backend
   ```

# 2. Встановіть необхідні залежності командою: 
   ```bash
   npm install
   ```

# 3. Створіть файл .env у папці backend/ та додайте наступну конфігурацію:
PORT=3001   

DATABASE_URL="file:./dev.db"

# 4. Налаштування бази даних (Prisma & SQLite). Виконайте міграцію для створення таблиць та запустіть сервер розробки командою: 
  ```bash
npx prisma db push
npm run dev
  ```

# (Бекенд-сервер успішно запуститься на порту http://localhost:3001)

#  Налаштування та запуск Фронтенду
   Перейдіть у папку з фронтендом та встановіть залежності командами: 
  ```bash
cd ../frontend
npm install
  ```

# Створіть файл .env.local у папці frontend/ для вказання адреси API:
NEXT_PUBLIC_API_URL=http://localhost:3001

# Запустіть фронтенд-сервер розробки командою: npm run dev
(Додаток буде доступний у вашому браузері за адресою http://localhost:3000)

## 📝 Як створити зразок тесту (Крок за кроком)
Відкрийте додаток у браузері (http://localhost:3000) та перейдіть на сторінку "Створення квізу".

Введіть Назву квізу (наприклад, "Основи JavaScript") та короткий Опис квізу (наприклад, "Тест на перевірку знань").

Додайте перше запитання та оберіть тип відповіді True / False:

Введіть текст запитання: "Масив в JS є об'єктом?"

Відмітьте правильний варіант: True (Правда).

Натисніть кнопку "+ Додати запитання" та оберіть тип Текстове поле:

Введіть текст запитання: "Який оператор використовується для строгого порівняння?"

У полі правильної відповіді вкажіть: ===

Додайте третє запитання з типом Чекбокси (Мультивибір):

Введіть текст запитання: "Оберіть типи даних в JavaScript"

Заповніть 4 варіанти відповідей у текстові інпути: String, Number, Boolean, HTML.

Поставте галочки навпроти правильних варіантів: відмітьте String, Number та Boolean.

Натисніть кнопку "Створити квіз". Додаток відвалідує дані, збереже їх у базу даних SQLite та автоматично перенаправить вас на сторінку зі списком усіх тестів.

## 🎨 Особливості реалізації та UI/UX
Оптимістичне та миттєве видалення: На сторінці списку тестів видалення відбувається моментально у локальному стейті клієнта, картка зникає з екрана без затримок та перезавантажень.

Повна архітектурна сумісність: Фронтенд самостійно трансформує складні UI-поля форми у плоску структуру (options та correctAnswer у вигляді масивів та рядків) за допомогою Zod перед відправкою, повністю захищаючи додаток від помилок сервера.

Режим "Лише для читання": На сторінці деталей тесту питання відображаються у фіксованому вигляді для перегляду структури квізу. Правильні відповіді безпечно та контрастно підсвічуються зеленим кольором.

Адаптивна верстка: Сітка списку тестів автоматично відображає рівно 4 картки в ряд на великих моніторах ПК, адаптується до 2-3 колонок на планшетах та перелаштовується в 1 зручну колонку на мобільних телефонах.
